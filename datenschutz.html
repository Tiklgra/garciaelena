3. Tus Derechos

Tienes derecho en todo momento a:

  • Obtener información sobre tus datos almacenados (Art. 15 RGPD)
  • Solicitar la corrección de datos incorrectos (Art. 16 RGPD)
  • Solicitar la eliminación de tus datos (Art. 17 RGPD)
  • Solicitar la restricción del procesamiento (Art. 18 RGPD)
  • Solicitar la portabilidad de datos (Art. 20 RGPD)
  • Oponerte al procesamiento (Art. 21 RGPD)
  • Revocar un consentimiento otorgado (Art. 7 párr. 3 RGPD)
  • Presentar una queja ante una autoridad supervisora (Art. 77 RGPD)

4. Recopilación de Datos en Este Sitio Web

Hosting

Este sitio web está alojado en un proveedor externo (hoster). Los datos personales recopilados en este sitio web se almacenan en los servidores del hoster. Esto puede incluir direcciones IP, solicitudes de contacto, metadatos, datos de comunicación, accesos al sitio web y otros datos generados a través del sitio web.

Archivos de Registro del Servidor

El proveedor de las páginas recopila y almacena automáticamente información en los llamados archivos de registro del servidor, que tu navegador transmite automáticamente:

  • Tipo y versión del navegador
  • Sistema operativo utilizado
  • URL de referencia
  • Nombre del host del ordenador que accede
  • Hora de la solicitud del servidor
  • Dirección IP

Contacto

Si nos contactas por correo electrónico, teléfono o formulario de contacto, tu consulta, incluyendo todos los datos personales resultantes (nombre, consulta, dirección de correo electrónico), se almacenará y procesará con el fin de gestionar tu solicitud.

5. Servicios Externos

Calendly

Utilizamos el servicio de reserva de citas Calendly (Calendly LLC, 3423 Piedmont Road NE, Atlanta, GA 30305, EE.UU.). Cuando reservas una cita, los datos que ingresas se transmiten y procesan en Calendly.

Más información: https://calendly.com/privacy

Testimonial.to

Para mostrar reseñas de clientes utilizamos el servicio Testimonial.to. Al cargar las reseñas, se pueden transmitir datos a Testimonial.to.

Google Fonts

Este sitio utiliza Google Fonts para una presentación uniforme de las fuentes. Las Google Fonts se cargan desde servidores externos.

6. Comunicación por WhatsApp

Ofrecemos la posibilidad de comunicarse a través de WhatsApp (Meta Platforms Ireland Limited). Si nos contactas a través de WhatsApp, tus datos se procesarán de acuerdo con las políticas de privacidad de WhatsApp.

Más información: Política de Privacidad de WhatsApp

7. Seguridad de Datos

Este sitio utiliza cifrado SSL/TLS por razones de seguridad y para proteger la transmisión de contenido confidencial. Puedes reconocer una conexión cifrada porque la barra de direcciones del navegador cambia de "http://" a "https://" y aparece un símbolo de candado.

8. Período de Retención

A menos que se especifique un período de retención más específico en esta política de privacidad, tus datos personales permanecerán con nosotros hasta que el propósito del procesamiento de datos deje de aplicarse.

9. Cambios en Esta Política

Nos reservamos el derecho de adaptar esta política de privacidad para que siempre cumpla con los requisitos legales actuales o para implementar cambios en nuestros servicios.
